## NAME
ramalama\-version - display version of RamaLama

## SYNOPSIS
**ramalama version**

## DESCRIPTION
Print version of RamaLama

## OPTIONS

#### **--help**, **-h**
Print usage message

## EXAMPLES

```
$ ramalama version
ramalama version 0.11.2
$ ramalama -q version
0.11.2
>
```
